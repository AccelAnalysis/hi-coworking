# Hi Coworking - Coming Soon

This is the coming soon page for Hi Coworking, a micro-coworking space.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React (Icons)
- Google Apps Script (Backend for form submissions)

## Setup

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the development server.

## Backend

The form submissions are handled by a Google Apps Script. The script code is located in `code.gs`.
Deploy the script as a Web App and update the `SCRIPT_URL` in `src/App.jsx`.
